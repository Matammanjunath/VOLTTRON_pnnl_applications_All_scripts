VOLTTRON Applications
=====================
This repository contains applications that revolve around following use cases

- Building Efficiency that help control building energy system performance

- Building-Grid Integration to support “beyond demand response” approach and integration of distributed energy resources” to grid.

- Simulation support for VOLTTRON applications
